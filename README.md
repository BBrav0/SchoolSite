# SchoolSite

A collection of web development projects and assignments from various courses, demonstrating proficiency in HTML, CSS, and JavaScript. This repository contains labs, assignments, and projects showcasing different web development concepts and techniques.

## Overview

This project serves as a portfolio of web development work completed across various courses. Projects range from basic HTML structure to interactive JavaScript applications, covering fundamental web development concepts and techniques.

## Project Structure

Projects are organized by course, with each course containing its respective labs, assignments, and projects:

```
public/
├── index.html                    # Main navigation page
└── [Course Name]/
    ├── Lab [N]/                  # Individual lab assignments
    ├── Assignment [N]/           # Individual assignments
    ├── Midterm/                  # Midterm projects
    ├── Final/                    # Final projects
    └── [other project files]     # Additional course materials
```

Each project may include:
- HTML files for structure
- CSS files for styling
- JavaScript files for interactivity
- Images, fonts, and other media assets
- Supporting documentation

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling, layout, and responsive design
- **JavaScript** - Interactive functionality and calculations
- **Web Fonts** - Custom typography (WOFF, TTF formats)

## Deployment

This project is configured for deployment on Vercel. The `vercel.json` file includes URL rewrites for cleaner navigation paths.

### Local Development

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd SchoolSite
   ```

2. Open `public/index.html` in a web browser, or use a local development server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server public
   ```

3. Navigate to `http://localhost:8000` in your browser
